# UC08 Dépendances inter‑projets

## Expression de besoin fonctionnelle
- **Objectifs** : cartographier dépendances, anticiper blocages, coordonner livrables.
- **Acteurs** : PMO, chefs de projet, équipes techniques.
- **Flux clés** : déclaration dépendance → validation → suivi dates → alertes → résolution.
- **Exceptions** : dépendances critiques escaladées en comité.
- **Critères d’acceptation** : vue réseau des dépendances, alertes proactives, historique décisions.

## Analyse ROI
- **Gains** : diminution des retards liés aux inter‑dépendances, meilleure coordination.
- **Coûts** : gouvernance de saisie, animation transverse.
- **KPI cibles** : nombre de dépendances en retard, délai moyen de résolution, % projets bloqués.
